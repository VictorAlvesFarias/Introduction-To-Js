Todos os meus projetos

Aqui você pode encontrar todos meus projetos baseados na sintaxe básica de Java Script.

Cronômetro

O Cronômetro foi feito usando as a função interna do Java Script chamada "Get Time", sua principal função e capturar o as informações de data do usuário, e com essa informação podemos construir varias lógicas, sendo uma delas um Relógio.

Unlock Content

Também usando o "Get Time", porem agora em conjunto com uma outra função, o Switch. Essa funcionalidade do Java Script traz a possibilidade de diversos escolhas de retorno para uma função de acordo com o variável ou valor setado de parâmetro.

Botões de Interação

Falando sobre o básico de Java Script, os botões os quais estamos falando são alguns construídos com o objetivo apresentar algumas interações bem básicas que a linguagem apresenta, como a alterações de classes, uso do console do navegar, Etc.

Calculadora

A Calculadora consiste numa função de inserção de strings no HTML que é a parte visual, e a inserção em um segundo input que é o que ira ser executado pela função Eval().

Verificação de Idade para Votar

Um pequeno exemplo de qual é o uso do If e Else, no caso ele é a função que guia qual ação deve ser seguida de acordo com certa condição, bem simples porem intuitivo.

Animação Variável de Estado

Também com If e Else, essas animações utilização dessas funções para definir qual parte da própria animação ele deve seguir no próximo loop, de acordo com com o estado que ela se encontra.

Efeito de Maquina de Escrever

Os objetos em Java Script são o que trazem os fluxos dos grandes projetos, essa pequena animação que faz um efeito de maquina de escrever, e graças a utilização de um objeto o código fica bem mais simples e organizado.

Repetição Controlada

O Repetidor utiliza-se das Funções While e For para criar um sistema de repetição de caracteres, onde sua função é inserir algum tipo de texto no HTML pré selecionado, você pode definir a quantidade de vezes que ele deve repetir e o texto que você escolher.

Monster in The Window

Esse projeto se trata de uma animação com Css e Java Script, onde o Java Script utiliza-se do For para fazer a mudança simultânea de diversas classes no HTML, para que haja uma alteração visual, gerando uma animação.
